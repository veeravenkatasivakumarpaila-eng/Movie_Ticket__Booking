
This project implements the core functionalities of a movie ticket booking platform:

✔ Users can log in and register

✔ Browse a list of movies

✔ View movie details and available shows

✔ Select seats and book tickets

✔ Generates ticket overview after booking

🚀 Technologies Used:

Java – Backend logic

Servlets – Handling web requests

HTML / CSS / JavaScript – Frontend UI

Web Server: Apache Tomcat or any servlet container

Optional: SQL scripts for database  setup
